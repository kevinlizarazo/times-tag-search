NYTimes Tag Search Engine




With the TimesTags API, you can mine the riches of the New York Times tag set. The TimesTags service matches your query to the controlled vocabularies that fuel NYTimes.com metadata. You supply a string of characters, and the service returns a ranked list of suggested terms.

The TimesTags API operates on a subset of Times tags: those tags used to create Times Topics pages. The subset includes approximately 27,000 tags:

  • (Des): 3,000 terms
  • (Geo): 1,500 terms
  • (Org): 7,500 terms
  • (Per): 15,000 terms

Problems searching? Try a page-specific hard refresh and try again (CTRL+Shift+R). Note: occasionally, API calls experience network lag.
